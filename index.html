Placeholder-Polyfill by jonahlyn

Dependencies

jQuery

Installation and Usage

1. Link to the Files

      
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js">
</script>
<script src="jquery.placeholderfix.js"></script>

2. Add Markup

      
<form>
    <label for="zip">Zip Code:</label>
    <input type="text" size="25" placeholder="XXXXX" name="zip" id="zip" />
    <input type="submit" value="Submit" />
</form>

3. Hookup the Plugin

  
<script>
  
$(function(){
  $(':text').placeholderFix();
});
  
</script>

4. Set Options if Needed

You can optionally set a default color for the placeholder text.

  
$(function(){
  $(':text').placeholderFix({'color':'#c9f0fd'});
});
